# Acknowledgments

SNAP (Spike Neuro Acquisition Platform) incorporates ideas, patterns, and
inspiration from the following open-source projects in the neuroscience
community:

## Foundation

**Open Ephys GUI** (GPL-3.0) — Foundation codebase
https://github.com/open-ephys/plugin-GUI
Developed by Josh Siegle, Jakob Voigts, and contributors at MIT,
Brown University, and the Allen Institute.

## Inspiration & Patterns

**Bonsai** (MIT) — Workflow UX patterns: node status indicators,
canvas annotations, searchable toolbox, reactive dataflow concepts
https://github.com/bonsai-rx/bonsai

**SpikeGLX** (Janelia Research Campus License 1.2) — Remote control
API protocol design, low-latency data fetch architecture
https://github.com/billkarsh/SpikeGLX

**SpikeInterface** (MIT) — Unified spike sorting framework,
RecordingExtractor/SortingExtractor abstraction
https://github.com/SpikeInterface/spikeinterface

**ProbeInterface** (MIT) — Probe geometry JSON format,
channel mapping standardization, probe library
https://github.com/SpikeInterface/probeinterface

**ONIX / Open Neuro Interface** (MIT) — Hardware abstraction layer,
unified clock synchronization architecture
https://github.com/open-ephys/onix-bonsai-onix1

**pyControl** (MIT) — State machine task definition syntax,
Python-on-microcontroller behavioral control patterns
https://github.com/pyControl/code

**DeepLabCut-Live** (LGPL-3.0) — Real-time markerless pose
estimation inference patterns
https://github.com/DeepLabCut/DeepLabCut-live

**OPETH** (GPL-3.0) — ZeroMQ bridge architecture reference
for Open Ephys to Python data streaming
https://github.com/open-ephys-plugins/zmq-interface

## Data Standards

**NWB (Neurodata Without Borders)** (BSD-3-Clause) — Neurophysiology
data standard
https://github.com/NeurodataWithoutBorders/pynwb

**DANDI Archive** (Apache-2.0) — Neurophysiology data sharing platform
https://github.com/dandi/dandi-archive
